## 解题思路

四种基本的遍历思想

前序遍历：根结点 —> 左子树 —> 右子树

  3. 根据中序遍历根节点的位置得到左子树size，在前序遍历中以此划分左右子树 。

  4. 前序遍历中划分的左右子树 的第一个值为根节点

  5. 递归求解上述过程。

9. 用两个栈实现队列

题目链接

牛客网

题目描述

用两个栈来实现一个队列，完成队列的 Push 和 Pop 操作。

解题思路

栈和队列属于两种不同的数据结构：

栈：“先进后出”，刪除和插入都在栈顶操作

队列：“先进先出”，在队头进行刪除操作，在队尾进行插入操作

本题目可以使用两个栈来完成队列操作：其中in 栈用来处理入栈（push）操作，out 栈用来处理出栈（pop）操作。一个元素进入 in 栈之后，出栈的顺序被反转。当元素要出栈时，需要先进入 out 栈，此时元素出栈顺序再一次被反转，因此出栈顺序就和最开始入栈顺序是相同的，先进入的元素先退出，这就是队列的顺序。


java

14. 剪绳子

Leetcode

题目描述

把一根绳子剪成多段，并且使得每段的长度乘积最大，其中n >= 2。

html

解题思路

贪心算法

所有数字n 都可以通过对一个因子 x 求整数部分 a，即（a = n / x） 和余数部分 b（ b = n % x）；

即得出数字 n 由 a 个 x 和 1 个 b 相加而成。

问题转化： 是否有优先级最高的因子 x 存在？若有，我们就可以把问题转化为求MAX( x^a * b) 。

当 n=2 时，n=1+1，1∗1<2，因此避免拆分出1 ；

当 n=3 时，n=1+2，1∗2<3，因此避免拆分出1；

当 n=4 时，n=2+2，2∗2=4，因此4可以拆分为2+2；

当 n=5 时，n=2+3，2∗3>5，因此5可以拆分为2+3；

当 n=5 时，n=2+3，2∗3>5，因此5可以拆分为2+3；

当 n=6 时，n=2+2+2=3+3，3∗3>2∗2∗2，因此拆分成3的优先级高于2；

……

易推出： 大数字都可以被拆分为多个小因子，以获取更大的乘积，只有 3 和 2 不需要拆分。优先拆分为3，如果剩余1，则将3+1拆成2+2。

算法流程：

当 n <= 3 时，直接返回 n - 1；
当 n > 3 时，求i，j使得 n = 3i+j ；
当 j == 0 时，返回 3^i；
当 j == 1 时，要将一个 1 + 3 转换为 2+2，返回 3^(i-1) * 2*2 ；
当 j == 2 时，返回 3^i * 2

java

题目描述

需要保证奇数和奇数，偶数和偶数之间的相对位置不变，这和书本不太一样。


## 解题思路

方法一：创建一个新数组，时间复杂度 O(N)，空间复杂度 O(N)。

方法二：使用冒泡思想，每次将当前偶数上浮到当前最右边。时间复杂度 O(N2)，空间复杂度 O(1)，时间换空间。
