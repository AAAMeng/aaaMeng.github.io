    2/27汇报内容

    基于深度学习的加密网络流量应用分类方法的研究与实现

    按照开题报告中的工作计划，已完成前期背景研究的调研，现阶段的工作是实现网络流量识别检测模型的核心功能并完成测试。

    首先是实验数据集，采集并标记了基于http协议的，ssh协议，流媒体，网络视频，即时通信等五种应用类型的流量数据，通过冗余清除、向量化、进制转换等预处理操作，分别获得了845，977，378，688，829个样本，该样本数据集将作为模型的输入用于训练和测试模型。

    然后是基于开题报告中的系统框架，设计了各个模块内的算法和模块间的交互过程并进行编码，开发了简单的原型模型实现各模块的核心功能。输入采集的样本数据集得到了分类的结果。

    目前的工作能够按照计划进行，接下来需要采集和标记其他类型应用的流量数据，制作用于最终模型测试的数据集，然后是确定评估指标，根据分类结果调整输入和模型参数，完善模型的所有功能。

    小论文结构

    Introduction

    传统的流量分类方法基于端口，DPI，加密流量和私有协议的出现导致传统方法不可行；基于流行为特征的方法，采用传统机器学习的方法存在特征提取的困难，本课题提出基于表征学习，挖掘流量数据的底层信息。传统的流量分类为异常检测二分类，实验中构造业务多分类。

    实验

    论文重读

    Paper1

    Malware Traffic Classification Using CNN for Representation Learning

    数据集： USTC-TFC2016

    使用2D-CNN，四种流量表示形式，用于二分类和多分类的应用场景

    仅对本课题提出的三种分类器进行对比实验(二分类，10分类，20分类)

    仅从早期检测、低误报率、协议独立、特征自提取、原始数据输入五个属性进行对比，没有量化的实验结果

    Limitation:

    1.没有CNN 模型调参

    2.缺少时间特征

    3.无法识别未知恶意流量

    Paper2

    End-to-end Encrypted Traffic Classification with One-dimensional CNN

    数据集：ISCX

    对6种VPN和6种non-VPN流量数据进行分类

    使用1D-CNN

    实验：

    1. 对比四种流量表示形式
    2. VPN和non-VPN的二分类，VPN的6分类，non-VPN的6分类，12分类
    3. 对比2D-CNN（量化结果）
    4. 和C4.5在precision和recall上对比，没有复现模型直接用了别人论文的数据

    Paper3

    HAST-IDS: Learning Hierarchical Spatial-Temporal Features Using DNN to Improve Intrusion Detection

    数据集：DARPA1998和ISCX2012

    提出了两个模型，一个仅使用CNN，一个使用CNN+RNN

    HAST1 : CNN first n byte/flow, m-dimention ONE-HOT编码 n*m 输入CNN（注：ONE-HOT编码会引入大量0填充，空间上的浪费）

    HAST2 : CNN+RNN r packet/flow, first q byte/packet, p-dimention ONE-HOT编码 r个p*q 输入CNN

    实验：

    基于两个数据对比了本课题模型和其他模型的结果(没有复现)

    Paper4

    Network Intrusion Detection: Based on Deep Hierarchical Network and Original Flow Data

    数据集：CICD2017+CTU dataset

    提出模型：CNN+LSTM, 10 packet/flow, 160 byte/packet, 1600输入到CNN,输出1600*1，划分成160*10输入到LSTM获得分类标签

    实验：

    1. 分别基于两个数据集进行实验，分别对比两个数据集其他模型的实验结果(没有复现)

    2. 对比本课题提出的CNN,LSTM,CNN+LSTM三种模型进行实验(从结果看出仅使用CNN均能达到0.98+, 在加上LSTM对结果没有显著提升，部分结果降低，zm认为CNN更好)

    3. 复现了statistical feature+半监督机器学习方法Kmeans+Decision tree

    4. 从时间上对比了KNN,NB,LR,RF,DT

    注：本文作者的论文PCCN中metric有借鉴价值，多分类的accuracy/precision/recall/F1
